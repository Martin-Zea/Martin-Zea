## 🚀 Proyectos Destacados

### 🏋️ GYM 2.0
> PWA mobile-first de seguimiento de entrenamiento con progresión asistida por IA

Aplicación web progresiva (offline-capable) para gestionar rutinas de gimnasio. Incluye seguimiento de series y pesos, temporizador de descanso, gráficas de progreso, soporte bilingüe (ES/EN) y recomendaciones generadas por LLMs (Groq + Cohere).

**Stack:** Angular 21 · TypeScript · SCSS · Groq AI · Cohere · PWA
